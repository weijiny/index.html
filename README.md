# videogamer
BETA


The goal of the game is for Green to catch Purple, while Purple tries to escape from Green.

**Control for GREEN**


**W** - move upward

**A** - move leftward

**S** - move downward

**D** - move rightward

**Control for PURPLE**


**P** - mov upward

**L** - move leftward

**;** - move downward

**'** - move rightward
